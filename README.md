# log-analysis-tool

# ログ解析ツール（C++）

## 概要
アクセスログを解析し、URLごとのアクセス数を集計してCSV形式で出力するツールです。

## 使用技術
- C++
- Linux

## 機能
- ログファイルの読み込み
- エラー行（ステータスコード400以上）の抽出
- URLごとのアクセス数集計
- CSVファイル出力

## 実行方法

### コンパイル
- g++ main.cpp -o log_tool

### 実行
- ./log_tool

## 入力データ（例）
- 192.168.1.1 2026-03-31 /index.html 200
- 192.168.1.2 2026-03-31 /login 404

## 出力結果（CSV）
- URL,Count
- /index.html,2
- /login,1

## 工夫した点
- mapを用いた効率的な集計処理
- stringstreamによるログ解析
- エラー処理の実装（ファイル未存在時など）
- 日付別集計
